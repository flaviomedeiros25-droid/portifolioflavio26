# Portfólio de Flávio Medeiros

Este é o meu portfólio pessoal, desenvolvido com HTML, CSS e JavaScript puro. O site apresenta minhas habilidades, projetos e informações de contato.

## Estrutura do Projeto

- `index.html` - Página principal do portfólio
- `css/style.css` - Estilos responsivos e modernos
- `js/script.js` - Funcionalidades interativas (menu mobile, scroll suave, etc.)
- `images/` - Pasta para imagens do portfólio

## Funcionalidades

- Design responsivo para desktop e mobile
- Menu de navegação com hamburger para dispositivos móveis
- Scroll suave entre seções
- Formulário de contato básico
- Animações de entrada nas seções

## Como Visualizar

Abra o arquivo `index.html` em qualquer navegador web moderno.

## Personalização

Para personalizar o portfólio:

1. Edite o conteúdo em `index.html`
2. Modifique os estilos em `css/style.css`
3. Adicione funcionalidades em `js/script.js`
4. Substitua as imagens em `images/`

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- Google Fonts (Roboto)
